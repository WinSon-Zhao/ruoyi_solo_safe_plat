# 阶段1：数据接入与日志解析

> **版本**：V1.0  
> **日期**：2026年5月  
> **目标**：安全设备日志能接进来、能解析正确、能存入 ES  
> **前置条件**：阶段0完成，运维验证平台可用

## 1. 阶段目标

### 1.1 核心目标

1. **设备管理**：支持多种安全设备的登记、配置、启停管理
2. **多协议接入**：支持 Syslog/HTTP/Kafka 三种接入方式
3. **日志解析**：支持 JSON/正则解析，字段映射，字典转换
4. **数据存储**：解析后的日志存入 EasySearch

### 3.2 parse 模块（数据解析）

**职责**：消费 raw topic，解析日志，字段映射，字典转换，写入 parsed topic

**核心组件**：

```java
// 解析规则引擎
@Component
public class ParseRuleEngine {
    
    public SecurityEvent parse(String rawLog, ParseRule rule) {
        // 1. 根据解析器类型选择解析方式
        ParsedResult result = switch(rule.getParserType()) {
            case JSON -> jsonParser.parse(rawLog, rule.getParserConfig());
            case REGEX -> regexParser.parse(rawLog, rule.getParserConfig());
            case GROK -> grokParser.parse(rawLog, rule.getParserConfig());
        };
        
        // 2. 字段映射
        Map<String, Object> mapped = fieldMapper.map(result.getFields(), rule.getFieldMappings());
        
        // 3. 字典转换
        Map<String, Object> standardized = dictConverter.convert(mapped, rule.getDictMappings());
        
        // 4. 构建 SecurityEvent
        return SecurityEvent.builder()
            .rawLog(rawLog)
            .parsedFields(standardized)
            .deviceId(rule.getDeviceId())
            .build();
    }
}

// 动态脚本解析（GraalJS）
@Component
public class ScriptParser {
    @Inject
    private ScriptEngine scriptEngine;
    
    public SecurityEvent parseWithScript(String rawLog, String scriptName) {
        // 调用 GraalJS 脚本进行自定义解析
        return scriptEngine.invoke("parse", rawLog);
    }
}
```

CREATE TABLE sec_device (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '主键',
    device_code VARCHAR(100) NOT NULL COMMENT '设备编码（唯一）',
    device_name VARCHAR(200) NOT NULL COMMENT '设备名称',
    device_type VARCHAR(50) NOT NULL COMMENT '设备类型:firewall/ids/ips/waf/edr/siem',
    vendor VARCHAR(100) COMMENT '厂商',
    model VARCHAR(100) COMMENT '型号',
    ip_address VARCHAR(50) COMMENT '设备IP',

    status TINYINT DEFAULT 1 COMMENT '状态:0停用 1启用',
    last_heartbeat DATETIME COMMENT '最后心跳时间',
    log_count BIGINT DEFAULT 0 COMMENT '累计接收日志数',
    
    -- 基础字段
