## 1. 阶段目标

### 1.1 核心目标

1. **安全规则引擎**：支持黑名单、威胁情报、漏洞匹配规则

4. **告警展示**：告警列表、统计图表、详情查看
5. **国标格式**：告警数据按 GB/T 44886 格式输出

### 4.2 规则引擎实现

```java
@Component
public class SecurityRuleEngine {
    
    @Inject
    private SecSecurityRuleMapper ruleMapper;
    @Inject
    private SecThreatIntelMapper intelMapper;
    
    /**
     * 执行规则匹配
     */
    public List<AlertHit> match(SecurityEvent event) {
        List<AlertHit> hits = new ArrayList<>();
        
        // 1. 获取启用的规则
        List<SecSecurityRule> rules = ruleMapper.selectEnabledRules();
        
        for (SecSecurityRule rule : rules) {
            boolean matched = switch (rule.getRuleType()) {
                case "blacklist_ip" -> matchBlacklistIp(event, rule);
                case "blacklist_port" -> matchBlacklistPort(event, rule);
                case "threat_intel" -> matchThreatIntel(event, rule);
                case "vuln_match" -> matchVulnerability(event, rule);
                case "signature" -> matchSignature(event, rule);
                default -> false;
            };
            
            if (matched) {
                hits.add(createAlertHit(event, rule));
            }
        }
        
        return hits;
    }
    
    private boolean matchBlacklistIp(SecurityEvent event, SecSecurityRule rule) {
        RuleConfig config = JSON.parseObject(rule.getRuleConfig(), RuleConfig.class);
        String fieldValue = getFieldValue(event, config.getMatchField());
        return config.getIps().contains(fieldValue);
    }
    
    private boolean matchThreatIntel(SecurityEvent event, SecSecurityRule rule) {
        RuleConfig config = JSON.parseObject(rule.getRuleConfig(), RuleConfig.class);
        String fieldValue = getFieldValue(event, config.getMatchField());
        return intelMapper.exists(config.getIntelType(), fieldValue);
    }
    
    private boolean matchSignature(SecurityEvent event, SecSecurityRule rule) {
        RuleConfig config = JSON.parseObject(rule.getRuleConfig(), RuleConfig.class);
        String fieldValue = getFieldValue(event, config.getMatchField());
        return Pattern.compile(config.getPattern()).matcher(fieldValue).find();
    }
}
```

## 5. 告警聚合设计

### 5.1 聚合逻辑

```java
@Component
public class AlertAggregator {
    
    /**
     * 聚合窗口时间（5分钟）
     */
    private static final long AGGREGATION_WINDOW_MS = 5 * 60 * 1000;
    
    /**
     * 聚合告警
     */
    public AlertGroup aggregate(List<SecurityAlert> alerts) {
        if (alerts.isEmpty()) return null;
        
        // 按聚合维度分组
        Map<String, List<SecurityAlert>> groups = alerts.stream()
            .collect(Collectors.groupingBy(this::getAggregationKey));
        
        List<AlertGroup> alertGroups = new ArrayList<>();
        
        for (Map.Entry<String, List<SecurityAlert>> entry : groups.entrySet()) {
            List<SecurityAlert> groupAlerts = entry.getValue();
            SecurityAlert firstAlert = groupAlerts.get(0);
            
            AlertGroup group = new AlertGroup();
            group.setGroupKey(entry.getKey());
            group.setSourceIp(firstAlert.getSourceIp());
            group.setAlertType(firstAlert.getAlertType());
            group.setAlertLevel(firstAlert.getAlertLevel());
            group.setTacticId(firstAlert.getTacticId());
            group.setTechniqueId(firstAlert.getTechniqueId());
            group.setAlertCount(groupAlerts.size());
            group.setFirstTime(firstAlert.getTimestamp());
            group.setLastTime(groupAlerts.get(groupAlerts.size() - 1).getTimestamp());
            group.setAlertIds(groupAlerts.stream()
                .map(SecurityAlert::getId)
                .collect(Collectors.toList()));
            
            alertGroups.add(group);
        }
        
        return alertGroups;
    }
    
    /**
     * 生成聚合键
     */
    private String getAggregationKey(SecurityAlert alert) {
        // 按时间窗口取整
        long windowStart = (alert.getTimestamp().getTime() / AGGREGATION_WINDOW_MS) 
            * AGGREGATION_WINDOW_MS;
        
        return String.format("%s|%s|%d",
            alert.getSourceIp(),
            alert.getAlertType(),
            windowStart
        );
    }
}
```

### 6.2 国标格式转换服务

```java
@Component
public class Gb44886Formatter {
    
    public GbAlertFormat format(SecurityAlert alert) {
        GbAlertFormat format = new GbAlertFormat();
        
        format.setEventType("alert");
        format.setEventTime(alert.getTimestamp());
        format.setEventId(alert.getId());
        
        // 数据源信息
        GbDataSource dataSource = new GbDataSource();
        dataSource.setDeviceId(alert.getDeviceId());
        dataSource.setDeviceType(alert.getDeviceType());
        dataSource.setVendor(alert.getVendor());
        format.setDataSource(dataSource);
        
        // 告警信息
        GbAlertInfo alertInfo = new GbAlertInfo();
        alertInfo.setAlertId(alert.getId());
        alertInfo.setAlertType(alert.getAlertType());
        alertInfo.setAlertName(alert.getAlertName());
        alertInfo.setAlertLevel(transAlertLevel(alert.getAlertLevel()));
        alertInfo.setAlertTime(alert.getTimestamp());
        alertInfo.setDescription(alert.getDescription());
        alertInfo.setSrcIp(alert.getSourceIp());
        alertInfo.setDstIp(alert.getDestinationIp());
        alertInfo.setDstPort(alert.getDestinationPort());
        alertInfo.setProtocol(alert.getProtocol());
        
        // ATT&CK 映射
        if (alert.getTacticId() != null) {
            alertInfo.setAttackTactic(getTacticName(alert.getTacticId()));
        }
        if (alert.getTechniqueId() != null) {
            alertInfo.setAttackTechnique(getTechniqueName(alert.getTechniqueId()));
            alertInfo.setAttackTechniqueId(alert.getTechniqueId());
        }
        
        format.setAlertInfo(alertInfo);
        return format;
    }
    
    private String transAlertLevel(String level) {
        return switch (level) {
            case "critical" -> "特别重大";
            case "high" -> "重大";
            case "medium" -> "较大";
            case "low" -> "一般";
            default -> "低";
        };
    }
}
```

## 9. 验收标准

### 9.1 功能验收

- [ ] 配置 IP 黑名单，命中后生成告警
- [ ] 配置威胁情报，命中后生成告警并标记来源
- [ ] 告警正确映射 ATT&CK 战术和技术
- [ ] 告警按 5 分钟窗口聚合，显示事件数
- [ ] 告警列表页面可查看聚合告警和原始告警
- [ ] 统计页面展示 TopN 攻击源、趋势图
- [ ] 国标格式接口可输出标准化告警数据
- [ ] skip_analysis 设备日志直接存 ES，不触发规则

### 9.2 性能验收

- [ ] 规则引擎吞吐量 > 5000 EPS
- [ ] 告警聚合延迟 < 1秒
- [ ] 告警查询响应 < 1秒（十万级数据）
