## 2. 安全事件设计

### 2.1 事件聚合逻辑

```java
@Component
public class EventAggregator {
    
    /**
     * 事件聚合窗口（30分钟）
     */
    private static final long EVENT_WINDOW_MS = 30 * 60 * 1000;
    
    /**
     * 聚合告警为安全事件
     */
    public SecurityEvent aggregate(List<AlertGroup> alertGroups) {
        if (alertGroups.isEmpty()) return null;
        
        // 按源IP和时间窗口分组
        Map<String, List<AlertGroup>> groups = alertGroups.stream()
            .collect(Collectors.groupingBy(this::getEventKey));
        
        List<SecurityEvent> events = new ArrayList<>();
        
        for (Map.Entry<String, List<AlertGroup>> entry : groups.entrySet()) {
            List<AlertGroup> groups = entry.getValue();
            AlertGroup first = groups.get(0);
            
            SecurityEvent event = new SecurityEvent();
            event.setEventKey(entry.getKey());
            event.setSourceIp(first.getSourceIp());
            
            // 计算事件等级（取最高）
            event.setEventLevel(calculateEventLevel(groups));
            
            // 4级分类（GB/T 37027）
            event.setEventClass(calculateEventClass(groups));
            
            // 统计信息
            event.setAlertCount(groups.stream().mapToInt(AlertGroup::getAlertCount).sum());
            event.setFirstTime(first.getFirstTime());
            event.setLastTime(groups.get(groups.size() - 1).getLastTime());
            
            // ATT&CK 攻击链
            event.setAttackChain(buildAttackChain(groups));
            
            // 关联告警
            event.setAlertGroupIds(groups.stream()
                .map(AlertGroup::getId)
                .collect(Collectors.toList()));
            
            events.add(event);
        }
        
        return events;
    }
    
    /**
     * 计算事件等级（GB/T 37027）
     * 一般/较大/重大/特别重大
     */
    private String calculateEventClass(List<AlertGroup> groups) {
        int criticalCount = countByLevel(groups, "critical");
        int highCount = countByLevel(groups, "high");
        int totalCount = groups.stream().mapToInt(AlertGroup::getAlertCount).sum();
        
        if (criticalCount >= 5 || totalCount >= 100) {
            return "特别重大";
        } else if (criticalCount >= 2 || highCount >= 10 || totalCount >= 50) {
            return "重大";
        } else if (highCount >= 3 || totalCount >= 20) {
            return "较大";
        } else {
            return "一般";
        }
    }
    
    /**
     * 构建攻击链
     */
    private List<AttackChainNode> buildAttackChain(List<AlertGroup> groups) {
        // 按 ATT&CK 战术排序
        return groups.stream()
            .sorted(Comparator.comparing(AlertGroup::getFirstTime))
            .map(g -> new AttackChainNode(
                g.getTacticId(),
                g.getTechniqueId(),
                g.getFirstTime(),
                g.getAlertType()
            ))
            .collect(Collectors.toList());
    }
}
```

### 3.2 资产-告警关联

```java
@Component
public class AssetAlertCorrelator {
    
    @Inject
    private SecAssetMapper assetMapper;
    
    /**
     * 关联告警与资产
     */
    public void correlate(SecurityAlert alert) {
        // 1. 源IP关联
        SecAsset sourceAsset = assetMapper.selectByIp(alert.getSourceIp());
        if (sourceAsset != null) {
            alert.setSourceAssetId(sourceAsset.getId());
            alert.setSourceAssetName(sourceAsset.getAssetName());
        }
        
        // 2. 目标IP关联
        SecAsset targetAsset = assetMapper.selectByIp(alert.getDestinationIp());
        if (targetAsset != null) {
            alert.setTargetAssetId(targetAsset.getId());
            alert.setTargetAssetName(targetAsset.getAssetName());
            alert.setTargetAssetImportance(targetAsset.getImportance());
        }
        
        // 3. 根据资产重要度调整告警等级
        if (targetAsset != null && "critical".equals(targetAsset.getImportance())) {
            alert.setAlertLevel(upgradeLevel(alert.getAlertLevel()));
        }
    }
}
```

### 4.2 处置执行引擎

```java
@Component
public class DisposalEngine {
    
    @Inject
    private SecDisposalDeviceMapper deviceMapper;
    @Inject
    private SecDisposalRecordMapper recordMapper;
    
    /**
     * 执行IP封堵
     */
    public DisposalResult blockIp(String ip, Long disposalDeviceId, Long eventId) {
        SecDisposalDevice device = deviceMapper.selectById(disposalDeviceId);
        
        // 创建处置记录
        SecDisposalRecord record = createRecord("block_ip", ip, device, eventId);
        
        try {
            // 调用设备API
            DisposalResult result = switch (device.getVendor()) {
                case "H3C" -> callH3CApi(device, "blockIp", ip);
                case "Huawei" -> callHuaweiApi(device, "blockIp", ip);
                case "Fortinet" -> callFortinetApi(device, "blockIp", ip);
                default -> throw new UnsupportedOperationException("不支持的设备厂商");
            };
            
            // 更新记录
            updateRecordSuccess(record, result);
            return result;
            
        } catch (Exception e) {
            updateRecordFailed(record, e);
            throw e;
        }
    }
    
    /**
     * 执行主机隔离
     */
    public DisposalResult isolateHost(String hostId, Long edrDeviceId, Long eventId) {
        SecDisposalDevice device = deviceMapper.selectById(edrDeviceId);
        
        SecDisposalRecord record = createRecord("isolate_host", hostId, device, eventId);
        
        try {
            DisposalResult result = callEdrApi(device, "isolate", hostId);
            updateRecordSuccess(record, result);
            return result;
        } catch (Exception e) {
            updateRecordFailed(record, e);
            throw e;
        }
    }
}
```

### 5.2 上报执行器

```java
@Component
public class ReportExecutor {
    
    @Inject
    private SecReportConfigMapper configMapper;
    @Inject
    private SecReportRecordMapper recordMapper;
    
    /**
     * 执行上报
     */
    public void executeReport(Long configId) {
        SecReportConfig config = configMapper.selectById(configId);
        
        // 1. 获取待上报数据
        List<?> data = fetchReportData(config);
        
        // 2. 格式转换
        List<?> formattedData = formatData(data, config.getFieldMapping());
        
        // 3. 构建请求
        HttpRequest request = buildRequest(config, formattedData);
        
        // 4. 发送上报
        HttpResponse response = request.execute();
        
        // 5. 记录结果
        saveReportRecord(config, data, response);
    }
    
    /**
     * 格式转换为国标
     */
    private List<Gb44886Format> formatToGb44886(List<SecurityAlert> alerts) {
        return alerts.stream()
            .map(this::convertToGbFormat)
            .collect(Collectors.toList());
    }
}
```

## 8. 验收标准

### 8.1 功能验收

- [ ] 多条关联告警自动聚合为安全事件
- [ ] 事件等级自动判定（一般/较大/重大/特别重大）
- [ ] 攻击链时间线正确展示
- [ ] 告警可一键封堵IP，防火墙执行成功
- [ ] 处置记录完整记录（指令/结果/时间）
- [ ] 告警通知发送到配置的渠道
- [ ] 数据可按国标格式上报到模拟监管平台

### 8.2 性能验收

- [ ] 事件聚合延迟 < 5分钟
- [ ] 处置命令执行 < 10秒
- [ ] 数据上报吞吐量 > 1000条/分钟
